# Руководство по поисковому запросу "Python" на Хабре

- По умолчанию запрос `Python` ищет точное совпадение по слову «Python». Если требуется охватить публикации, связанные с программированием на Python, но не содержащие слово «Python» в заголовке, рекомендуется использовать расширенный запрос:
  - `"Python" OR "Python programming" OR "разработка на Python" OR "питон"`
- При поиске статей за короткий период (например, за 7 дней) добавляйте уточнение по дате, чтобы избежать возвращения старых материалов, если API поддерживает фильтрацию.
- Если `search_habr` возвращает менее 3 статей, попробуйте один из следующих вариантов:
  1. Расширьте период (например, до `14d` или `1m`).
  2. Уточните запрос, добавив синонимы или связанные термины (см. выше).
  3. Уменьшите ограничение `limit` до 5, чтобы получить более релевантные результаты.

Эти рекомендации помогут получить более полные и релевантные наборы статей при составлении дайджеста по теме Python.
